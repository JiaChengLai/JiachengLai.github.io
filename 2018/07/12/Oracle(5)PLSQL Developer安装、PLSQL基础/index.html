PL/SQL编程语言

什么是PL/SQL

PL/SQL(Procedure Language/SQL)
PLSQL是Oracle堆sql语言的过程化扩展，指在SQL命令语言中添加了过程处理语句（如分支、循环），使SQL语言具有过程处理能力。把SQL语言的数据操纵能力与过程语言的数据处理能力结合起来，使得PLSQL面向过程但比过程语言简单、高效、灵活和实用。

PL/SQL中的游标(CURSOR)

在写java程序中有集合的概念，那么在PL/SQL中也会用到多条记录，这时候我们就要用到游标(CURSOR),游标可以储存查询返回的多条记录，类似于JAVA中的RESULTSET。

CURSOR的语法

在PL/SQL的declare中定义
CURSOR 游标名 [{参数名 数据类型，参数名 数据类型,....}] IS SELECT 语句;
例如
CURSOR c1 IS SELECT ename FROM emp;

游标的使用步骤

在PL/SQL的begin和end之间完成
1、打开游标
open c1;(打开游标执行查询)
2、取一行游标的值
fetch c1 into pjob;(取一行值到变量之中)
3、关闭游标
close c1;(关闭游标释放资源)
4、游标的结束方式
exit when c1%notfound;

注意:上面的pjob必须与emp表的job列类型一致;
定义:pjob emp.job%type

游标的属性

%isopen 游标是否打开
%rowcount 游标影响的行数
%found 游标有下一个
%notfound 游标没有下一个了

PL/SQL中的例外(EXCEPTION)

EXCEPTION的类型

no_data_found 没有找到数据
too_many_rows select..into语句匹配多个列
zero_divide 被0除
value_error 算数或转换错误
timeout_on_resource 在等待资源时发生超时(死锁问题)

EXCEPTION的处理

与java的例外处理机制不同，java的虚拟机会catch住所有的例外，而PL/SQL不会，所以我们在写PL/SQL时应处理好所有的例外。
